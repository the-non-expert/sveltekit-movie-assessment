# Project Overview

## Goal
Build a **movie listing web application** using SvelteKit to demonstrate proficiency in Svelte 5, TypeScript, Tailwind v4, and API integration for a job assessment.

## Core Purpose
A dark-themed movie discovery platform where users can:
- Browse and search movies from IMDB API
- Filter by genre, year, and rating
- Manage personal watchlists
- Authenticate with email/password

## Tech Stack
- **Framework**: SvelteKit (Svelte 5)
- **Styling**: Tailwind CSS v4 using Vite Plugin
- **Language**: TypeScript
- **Database**: Supabase (PostgreSQL)
- **API**: IMDB API
- **State Management**: Svelte stores (runes + custom stores)

## Key Constraints
- No Supabase Auth (custom implementation)
- Client-side filtering only
- Session persistence (24-hour auto logout)
- Well-documented, organized codebase
- Dark theme with distinctive typography

## Success Metrics
- All 6 core features working correctly
- Clean code architecture
- Proper use of Svelte 5 concepts (stores, binding, async/await, slots)
- IMDB API integration
- Comprehensive documentation
